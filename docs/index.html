Quantitative ECA Research

Minimal cellular automata, measured as a reproducible rule space.

Grav is a compact research repository focused on all 256 elementary cellular automata: reproducible metrics, clustering in feature space, and targeted search for localized structures such as translating fronts.

Read Project Overview Open Preprint

What the repository does

The current pipeline evaluates every ECA rule with entropy, perturbation sensitivity, a Lyapunov-like growth heuristic, and a coarse glider indicator, then studies the geometry of rule space with clustering and stability checks.

What makes it useful

The project treats rule numbers as points in a metric space rather than as isolated pictures. That shifts the emphasis from visual folklore to reproducible, comparable observables.

Secondary research line

Separate scripts explore domains, interfaces, and translating defects, with the Rule 69 line currently being the strongest localized-structure case study in the repository.

Key Results

Current research claims

Full 256-rule sweep

All elementary cellular automata are analyzed under one shared numerical protocol.

Metric rule-space map

Rules are compared through entropy, sensitivity, and related observables.

Cluster stability checks

Silhouette-based model choice is paired with ARI stability and permutation testing.

Localized structures

Rule-specific search scripts investigate domains, defects, interfaces, and fronts.

Reproducibility

Run the core pipeline

python3 -m pip install -r requirements.txt
make metrics
make cluster
make test

The canonical package entrypoints are python3 -m grav.metrics and python3 -m grav.cluster. Research scripts for fronts and particle search remain available in grav/research/.
