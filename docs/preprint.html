Preprint

Metric structure of elementary cellular automata rule space

This page condenses the current repository preprint into a web-readable form. The focus is a reproducible metric treatment of all 256 ECA rules.

Abstract

Grav studies the full ECA family through entropy, perturbation sensitivity, and a Lyapunov-like observable. Rules are then clustered in feature space with KMeans, the number of clusters is selected by silhouette score, and stability is checked with adjusted Rand index and a permutation test.

The current result is a visibly structured partition in the two-dimensional projection (H, S), but the chosen null model does not yet support a strong statistical claim that the geometry is deeper than the marginal feature distributions.

Setup

Each rule evolves on a periodic ring of size N. For each rule, multiple random initial states are simulated, and the resulting trajectories are summarized by reproducible numerical observables.

Main observables

  • Entropy: binary Shannon entropy averaged over time and runs.
  • Sensitivity: mean normalized Hamming distance between perturbed trajectories.
  • Lyapunov-like growth: logarithmic growth estimate for perturbation size.
  • Glider heuristic: a coarse marker for localized translating behavior.

Clustering result

In the repository's current workflow, three clusters emerge as the strongest silhouette solution in the standard (entropy_mean, sens_mean) projection. Across initialization seeds, the partition is algorithmically stable.

Interpretation limit

Stability alone is not enough. The permutation test keeps the marginals of the features but destroys their joint structure, and under that null model the present feature set does not yet justify a hard claim of statistically significant phase structure.

Why the project still matters

Even with that limitation, the repository already provides a useful foundation: a reproducible baseline for comparing rules, refining observables, testing symmetries, and eventually tying metric organization to actual structural phenomena such as interfaces and particles.
