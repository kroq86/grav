Case Study

Ballistic interfaces in Rule 69

The strongest localized-structure story in the repository is currently the Rule 69 line: periodic domains, finite defects between them, and exact translating interfaces.

Claim

For the currently studied fixed-point domains of Rule 69, the repository reports computer-assisted witnesses for interfaces that move ballistically with velocity -1. In the usual notation, the target condition is F(x) = sigma^-1(x).

How the witnesses are checked

  • A two-domain background is built on a large periodic ring.
  • A finite defect is embedded near the interface cut.
  • The next state is checked against an exact spatial shift.
  • Locality is verified both before and after the update.

Why this matters

This is the point where the repository stops being only a metric survey and starts moving toward a structural theory of localized objects in discrete dynamical media. Rule 69 is therefore the best candidate for a future polished note with certificates, diagrams, and a more formal symbolic-dynamics interpretation.

Current status

The evidence here is computational and bounded, not yet a finished general theorem. But it gives the repository a concrete second pillar beyond the global 256-rule sweep.
