# Метрическая структура пространства правил элементарных клеточных автоматов

## Аннотация

В работе проводится количественный анализ всех 256 элементарных клеточных автоматов (ECA) на основе набора воспроизводимых численных метрик динамики. Исследуется геометрическая структура пространства правил в координатах средней энтропии, чувствительности к возмущению и Lyapunov-подобной оценки. Выполняется кластеризация правил методом KMeans с выбором числа кластеров по коэффициенту силуэта и оценкой устойчивости через Adjusted Rand Index. Дополнительно проводится пермутационный тест статистической значимости наблюдаемой структуры.

Получено устойчивое алгоритмическое разбиение правил на три группы в двумерной проекции (H, S). Однако пермутационный тест не отвергает нулевую гипотезу о случайности структуры при сохранении маргинальных распределений признаков. Результаты указывают на наличие выраженной геометрии данных, но недостаточность текущего набора признаков для строгого статистического вывода о фазовой структуре пространства правил.

---

## 1. Введение

Элементарные клеточные автоматы (ECA) представляют собой класс дискретных динамических систем, задаваемых локальным бинарным правилом радиуса 1 на одномерной решётке. Пространство из 256 правил демонстрирует широкий спектр режимов — от фиксированных до хаотических.

Классическая классификация Вольфрама носит качественный характер. В данной работе предпринимается попытка построить количественную метрическую картину пространства правил и исследовать существование устойчивой геометрической структуры без визуальной интерпретации.

---

## 2. Формальная постановка

Пусть задана решётка размера N с периодическими граничными условиями.  
Состояние системы в момент времени t обозначается:

x(t) ∈ {0,1}^N

Эволюция определяется локальным правилом:

x_i(t+1) = f_r(x_{i-1}(t), x_i(t), x_{i+1}(t))

где r ∈ {0, …, 255} — номер правила.

Для каждого правила проводится усреднение по нескольким случайным начальными условиям.

---

## 3. Метрики динамики

### 3.1 Средняя энтропия

Для каждой конфигурации вычисляется бинарная энтропия:

H = -p log2 p - (1 - p) log2(1 - p)

где p — доля единиц.

Итоговая величина H_mean — усреднение по времени и прогонам.

---

### 3.2 Чувствительность к возмущению

Для оценки чувствительности:

1. Формируется конфигурация x.
2. Создаётся возмущённая версия x', отличающаяся одним битом.
3. Оба состояния эволюционируют параллельно.
4. Измеряется нормированное расстояние Хэмминга d(t).

S_mean — среднее значение d(t) по времени и прогонам.

---

### 3.3 Lyapunov-подобная оценка

Вычисляется:

λ = (1/T) * log(d(T) / d(0))

где d(t) — расстояние между траекториями.

Метрика служит численной оценкой экспоненциального роста различий.

---

### 3.4 Эвристика локальных структур

Проводится тест на сдвиговую периодичность при локальном начальном условии.  
Метрика используется как индикатор возможного существования устойчивых движущихся структур.

---

## 4. Пространство признаков

Каждое правило отображается в точку:

(H_mean, S_mean, λ_mean, дополнительные статистики)

Основной анализ проводится в двумерной проекции (H, S).

---

## 5. Кластеризация

### 5.1 Метод

Используется алгоритм KMeans.

Число кластеров k выбирается по максимуму коэффициента силуэта.

### 5.2 Результат

Максимум силуэта достигается при:

k = 3

Silhouette ≈ 0.74

### 5.3 Устойчивость

Оценка устойчивости проводится через Adjusted Rand Index (ARI) при различных инициализациях.

Получено:

ARI ≈ 1.0

Это указывает на алгоритмическую устойчивость разбиения.

---

## 6. Пермутационный тест

Для проверки статистической значимости выполняется перестановочный тест:

1. Независимая перестановка координат признаков.
2. Пересчёт силуэта.
3. Оценка эмпирического p-значения.

Получено:

p ≈ 0.45

Следовательно, нулевая гипотеза о случайной структуре при сохранении маргинальных распределений не отвергается.

---

## 7. Интерпретация

Наблюдается разделение правил на три группы:

- Низкая энтропия / низкая чувствительность  
- Высокая энтропия / высокая чувствительность  
- Промежуточные режимы  

Разбиение устойчиво к инициализации алгоритма, однако статистическая проверка не подтверждает его значимость в текущем пространстве признаков.

Это означает, что наблюдаемая геометрия может быть следствием распределения самих метрик, а не наличия глубинной фазовой структуры.

---

## 8. Ограничения

- Анализ преимущественно двумерен.
- Метрики являются численными эвристиками.
- Пермутационный тест полностью разрушает корреляционную структуру.
- Не учитываются симметрии правил (отражение, комплементарность).
